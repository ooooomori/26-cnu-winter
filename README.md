# 프로젝트 소개
2026년 겨울방학 서강대학교 CNU 입문 프로젝트입니다.

## 🔍 주요 기능
 - 단어 뜻 및 활용 가능한 예문 검색
 - LLM을 이용해 단어별 학습용 예문 생성
 - 나만의 단어장에 검색한 단어와 예문 저장
 - 단어장에 단어 및 예문 추가, 삭제, 편집 등
 - 단어장에 추가한 단어로 간단한 복습용 문제 풀기
